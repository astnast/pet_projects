# Репозиторий учебных проектов по бизнес-системному анализу (BSA)

Это репозиторий учебных проектов по бизнес-системному анализу. Здесь собраны артефакты, демонстрирующие полный цикл работы аналитика: от выявления требований и моделирования процессов до проектирования данных, интерфейсов и определения нефункциональных требований (NFR).

Все работы выполнены в рамках углубленного изучения методологии BSA на примере двух кейсов:
1. Sign up for a haircut (HCA) — Система онлайн-записи для сети барбершопов.
2. Delivery of orders (DOO) — Система доставки заказов курьерской службы.

# Навыки и Инструменты изученные в рамках этих кейсов:

| Категория | Инструменты и методологии |
|-----------|---------------------------|
| Сбор требований | Интервью, мозговой штурм, User Stories, Use Cases |
| Моделирование процессов | BPMN 2.0, IDEF0, Диаграммы потоков данных (DFD) |
| Моделирование данных | ER-диаграммы, UML Class Diagram, SQL (логическая модель) |
| Проектирование UI/UX | Wireframing, Описание экранных форм, Сценарии использования |
| Управление требованиями | Глоссарий, Матрица CRUD, Статусные модели, Rights Matrix |
| Нефункциональные требования | Performance, Security, Scalability, Availability, Reliability |
| Документация | Vision, Техническое задание, Спецификации требований |

## Структура репозитория

```
src/
├── Sign_up_for_a_haircut/   — артефакты по задаче «Запись на стрижку» (барбершоп)
├── Delivery_of_orders/   — артефакты по задаче «Доставка заказов» (курьерская служба)
└── other/ — общие упражнения по методологии анализа
```
